# Smart Study Website

## Overview

Smart Study Website is an AI-powered web application designed to help students study more effectively, plan their time, and track their performance. It uses modern AI tools to make learning smarter, faster, and more personalized.

---

## Features

### AI Notes Summarizer

Users can enter their notes, and the AI will convert them into short, clear, and easy-to-understand summaries.

### Smart Study Planner

Users can enter their subjects and available time, and the system will automatically generate an optimized study schedule.

### Performance Tracker

Users can track:

* test scores
* study hours
* overall progress

The data is shown using simple graphs so users can easily see their improvement.

### AI Chatbot (Doubt Solver)

A built-in AI chatbot that:

* answers questions
* explains concepts
* provides personalized guidance

## How It Works

1. User signs up or logs in
2. Accesses the dashboard
3. Adds notes and gets AI-generated summaries
4. Creates a study plan
5. Tracks daily progress
6. Uses the AI chatbot to solve doubts
